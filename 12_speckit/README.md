# Todolist (Streamlit, SQLite3)

Aplikacja Todolist: dodawanie zadań (tytuł, opcjonalny opis), oznaczanie jako ukończone, usuwanie. Lista z filtrem (wszystkie / aktywne / ukończone). Dane w pliku `tasks.db`.

## Uruchomienie

1. Przejdź do katalogu aplikacji: `cd 12_speckit`
2. Aktywuj istniejące venv (np. `source .venv/bin/activate`)
3. Uruchom: `streamlit run src/app.py`
4. Otwórz w przeglądarce adres z terminalu (np. http://localhost:8501)

Nie tworzymy venv ani nie instalujemy zależności — używamy istniejącego środowiska.
